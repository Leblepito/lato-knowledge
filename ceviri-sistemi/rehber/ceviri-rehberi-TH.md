# 🌐 บอทแปลภาษา Lato — คู่มือการใช้งาน

> **บอท:** @Latotry_bot
> **ตำแหน่ง:** 🌐 หัวข้อแปลภาษา (Topic 146)
> **ภาษา:** 🇹🇭 ไทย ↔ 🇹🇷 ตุรกี ↔ 🇬🇧 อังกฤษ

---

## 📌 บอทนี้ใช้ทำอะไร?

บอทแปลภาษา Lato ช่วยให้พนักงานโรงแรมสื่อสารกันได้ทันทีในภาษาต่างๆ เมื่อมีคนส่งข้อความเสียงหรือข้อความตัวอักษร บอทจะ:

1. **ตรวจจับภาษาอัตโนมัติ** (ไทย ตุรกี หรืออังกฤษ)
2. **แปลข้อความ** เป็นภาษาอื่นๆ
3. **แสดงผลการแปล** ในกลุ่มให้ทุกคนเห็น
4. หากคุณลงทะเบียนเสียงแล้ว บอทสามารถ **พูดการแปลด้วยเสียงของคุณ** ได้

**ตัวอย่าง:**
> อห์เหม็ดส่งข้อความเสียงภาษาตุรกี → บอทแปลเป็นไทยและอังกฤษ → พนักงานที่พูดไทยและอังกฤษเข้าใจทันที

---

## 🚀 วิธีใช้งาน

### ส่งข้อความเสียง
1. ไปที่หัวข้อ 🌐 **แปลภาษา**
2. กดไอคอนไมโครโฟonของ Telegram แล้วพูด
3. บอทจะฟัง ตรวจจับภาษา และแปลให้

### ส่งข้อความตัวอักษร
1. ไปที่หัวข้อ 🌐 **แปลภาษา**
2. พิมพ์ข้อความของคุณ
3. บอทจะตรวจจับภาษาและแปลให้

> 💡 **เคล็ดลับ:** พูดตามปกติให้ชัดเจน ไม่ต้องพูดช้าเกินไป

---

## 🎙️ การลงทะเบียนเสียง

เพื่อให้บอทพูดการแปล **ด้วยเสียงของคุณ** คุณต้องลงทะเบียนเสียง

### ขั้นตอนการลงทะเบียน:

| ขั้นตอน | สิ่งที่ต้องทำ |
|---------|-------------|
| **1** | ส่งคำสั่ง `/register` ในหัวข้อ 🌐 แปลภาษา |
| **2** | บอทจะถามชื่อของคุณ — พิมพ์ชื่อหรือชื่อเล่น (เช่น `สมชาย`) |
| **3** | บอทจะขอตัวอย่างเสียง — พูดตามปกติประมาณ **15–30 วินาที** |

### พูดอะไรในตัวอย่างเสียง?

อะไรก็ได้! นี่คือตัวอย่าง:

> *"สวัสดีครับ ผมชื่อสมชาย ผมทำงานที่โรงแรมลาโต้ในตำแหน่งพนักงานต้อนรับ งานประจำวันของผมคือต้อนรับแขก จัดการการจองห้อง และรับโทรศัพท์ ผมรักงานของผมและพยายามให้บริการที่ดีที่สุดแก่แขกของเรา"*

**ข้อควรระวัง:**
- ✅ พูดให้สงบและชัดเจน
- ✅ บันทึกในที่เงียบ ไม่มีเสียงรบกวน
- ✅ พูดอย่างน้อย 15 วินาที ไม่เกิน 30 วินาที
- ❌ อย่ากระซิบ
- ❌ อย่าพูดเร็วเกินไป

## 🧬 เสียงลอกแบบทำงานอย่างไร

บอทใช้เทคโนโลยี **ElevenLabs** ในการลอกแบบเสียงของคุณ:

1. เมื่อคุณส่งตัวอย่างเสียง ระบบจะเรียนรู้ลักษณะเสียงของคุณ
2. จากนั้น เมื่อบอทแปลในนามของคุณ บอทจะ **พูดด้วยเสียงของคุณ**
3. ตัวอย่าง: เมื่อพนักงานไทยพูดภาษาไทย บอทจะแปลเป็นตุรกีและอ่านการแปล **ด้วยเสียงของคุณ**

> 🔒 เสียงของคุณใช้เฉพาะในบอทนี้เท่านั้น ไม่ได้แชร์กับบริการอื่นใด

## 🔤 การแก้ไขการออกเสียง (Accent Corrections)

บางครั้งบอทอาจได้ยินชื่อเฉพาะหรือคำท้องถิ่นผิด คุณสามารถสอนการออกเสียงที่ถูกต้องได้:

### คำสั่ง:
```
/accent add คำผิด คำที่ถูก
```

### ตัวอย่าง:
```
/accent add phuket puket
/accent add sawatdee savatdi
/accent add kapib kapik
```

วิธีนี้จะทำให้บอทออกเสียงคำเหล่านี้ถูกต้องทุกครั้ง

### ดูการแก้ไขที่มีอยู่:
```
/accent list
```

### ลบการแก้ไข:
```
/accent remove คำผิด
```

## 📋 รายการคำสั่ง

| คำสั่ง | คำอธิบาย |
|--------|----------|
| `/register` | ลงทะเบียนเสียง — บอทจะเรียนรู้เสียงของคุณ |
| `/myvoice` | แสดงสถานะการลงทะเบียนเสียงของคุณ (ลงทะเบียนแล้ว พร้อมใช้งาน) |
| `/accent` | จัดการการแก้ไขการออกเสียง (`/accent add`, `/accent list`, `/accent remove`) |
| `/languages` | แสดงรายการภาษาที่บอทรองรับ |
| `/speakers` | แสดงรายชื่อผู้ที่ลงทะเบียนเสียงแล้ว |
| `/help` | แสดงคำสั่งทั้งหมดและข้อมูลการใช้งาน |

## ❓ คำถามที่พบบ่อย

**ถาม: บอทแปลทุกข้อความเป็นเสียงหรือไม่?**
ตอบ: ไม่ บอทจะให้การแปลเป็นตัวอักษรก่อน หากคุณลงทะเบียนเสียงแล้วและเหมาะสม บอทจะให้เสียงด้วย

**ถาม: ลงทะเบียนเสียงได้หลายภาษาหรือไม่?**
ตอบ: ในขณะนี้ การลงทะเบียนเสียงครั้งเดียวต่อคนก็เพียงพอ บอทจะนำเสียงของคุณไปใช้กับทุกภาษา

**ถาม: ฉันอยากอัปเดตตัวอย่างเสียง ต้องทำอย่างไร?**
ตอบ: ใช้คำสั่ง `/register` อีกครั้ง การบันทึกใหม่จะแทนที่อันเก่า

**ถาม: บอทแปลผิด ต้องทำอย่างไร?**
ตอบ: ลองใช้คำสั่ง `/accent` เพื่อแก้ไขการออกเสียง
