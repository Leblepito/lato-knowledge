## Kısaltmalar
- AC = แอร์ (Air Conditioner)
- VRF = Variable Refrigerant Flow (ระบบทำความเย็นแปรผัน)
- HVAC = ระบบทำความร้อน ระบายอากาศ และปรับอากาศ
- R410A = แก๊สทำความเย็นชนิดหนึ่ง
- R32 = แก๊สทำความเย็นรุ่นใหม่
- WC = ห้องน้ำ (Water Closet)
- PPR = ท่อพลาสติกพอลิโพรพิลีน
- PEX = ท่อยืดหยุ่นพอลิเอทิลีน
- PVC = ท่อพลาสติกพีวีซี

## Yaygın Hatalı Yazılışlar / Noisy Input
- "แอร์ไม่เย็น" → ใช้: "แอร์ไม่ทำความเย็น" (gaz eksik/kompresör)
- "แอร์เสีย" → ใช้: "แอร์ขัดข้อง" (บอกประเภทเสีย)
- "เติมแก๊ส" → ใช้: "เติมแก๊สทำความเย็น" (R410A/R32)
- "น้ำหยด" → ใช้: "แอร์หน่วยในหยดน้ำ" หรือ "ท่อน้ำรั่ว"
- "split" → ใช้: "แอร์แยกส่วน" (split AC)
- "คอมเพรสเซอร์ดัง" → ใช้: "คอมเพรสเซอร์มีเสียงผิดปกติ"
- "ก๊อกเสีย" → ใช้: "ก๊อกน้ำต้องซ่อม"
- "เครื่องทำน้ำอุ่นไม่ทำงาน" → ใช้: "เครื่องทำน้ำร้อนไม่ทำงาน"
- "ห้องน้ำอุดตัน" → ใช้: "ห้องน้ำ/โถส้วมอุดตัน"
- "สีลอก" → ใช้: "สีทาอาคารลอก/ด่าง"
- "กลอนเสีย" → ใช้: "กลอนประตูขัดข้อง"
- "รั่ว" → บอก: "ท่อน้ำรั่ว" หรือ "แก๊สทำความเย็นรั่ว"
- "ท่อแตก" → ใช้: "ท่อน้ำแตก (ฉุกเฉิน)"
