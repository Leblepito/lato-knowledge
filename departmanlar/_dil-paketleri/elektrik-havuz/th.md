# ไฟฟ้าและสระว่ายน้ำ — ภาษาไทย Terminology

## Kısaltmalar
- RCD = เซฟตี้คัท (Safety cut) — อุปกรณ์ป้องกันไฟรั่ว
- ATS = สวิตช์สับไฟอัตโนมัติ (Automatic Transfer Switch)
- SPD = อุปกรณ์ป้องกันไฟเกิน (Surge Protective Device)
- VFD = อินเวอร์เตอร์ (Variable Frequency Drive)
- kWh = หน่วยไฟฟ้า (กิโลวัตต์-ชั่วโมง)
- kVA = กิโลโวลต์-แอมแปร์
- PFC = ระบบชดเชยไฟฟ้า (Power Factor Correction)
- MCB = เบรกเกอร์ขนาดเล็ก (Miniature Circuit Breaker)
- TDS = ปริมาณของแข็งละลายรวม (Total Dissolved Solids)
- ORP = ค่าออกซิเดชัน-รีดักชัน (Oxidation-Reduction Potential)

## Yaygın Hatalı Yazılışlar / Noisy Input
- "ตู้ไฟ" → ใช้: "ตู้ไฟฟ้า" (กรณีเขียนสั้น)
- "เซฟตี้" → ใช้: "เซฟตี้คัท" หรือ "RCD"
- "ปั๊ม" → ใช้: "ปั๊มน้ำสระ" (บอกประเภทชัดเจน)
- "คลอรีนต่ำ" → ใช้: "ค่าคลอรีนต่ำ"
- "ph สูง" → ใช้: "ค่า pH สูง"
- "ล้างไส้กรอง" → ใช้: "ล้างกรองย้อนกลับ" (backwash)
- "มอเตอร์ไม่ทำงาน" → บอก: "ปั๊มน้ำสระไม่ทำงาน" หรือ "มอเตอร์ไฟฟ้าไม่ทำงาน"
- "ไฟตก" → ใช้: "ไฟดับ" หรือ "แรงดันไฟต่ำ" (specify)
- "ไฟรั่ว" → ใช้: "เซฟตี้คัทตัด" (RCD tripped)
- "เกลือ" → ใช้: "เครื่องผลิตคลอรีนจากเกลือ" (salt chlorinator)
- "อินเวอร์เตอร์เสีย" → ใช้: "อินเวอร์เตอร์ VFD เสีย"
- "สายดินไม่ดี" → ใช้: "ระบบสายดินไม่ได้มาตรฐาน"
