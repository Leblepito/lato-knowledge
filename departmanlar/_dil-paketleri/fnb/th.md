## Kısaltmalar
- **F&B** — Food & Beverage (อาหาร & เครื่องดื่ม)
- **HACCP** — Hazard Analysis Critical Control Points
- **FIFO** — First In First Out (เข้าก่อนออกก่อน)
- **COGS** — Cost of Goods Sold
- **POS** — Point of Sale
- **BEO** — Banquet Event Order

## Yaygın Hatalı Yazılışlar / Noisy Input
- "คัว" → ครัว
- "เตาว" → เตาอบ
- "หมอทอด" → หม้อทอดน้ำมัน
- "ตู้เยน" → ตู้เย็น
- "ตู้แขง" → ตู้แข็ง
- "อาหารปลอดภัย" → ความปลอดภัยอาหาร
- "hacp" / "hccp" → HACCP
- "อุณหภูม" → บันทึกอุณหภูม
- "วันหมด" / "exp" → วันหมดอายุ
- "บุฟเฟต์" → บุฟเฟต์น้ำชามร์
- "เมนู" → เมนู

1. "ครัวต้องทำความสะอาดทุกเช้าและเย็น"
