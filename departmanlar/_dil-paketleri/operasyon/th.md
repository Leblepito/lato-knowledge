# Operasyon — ภาษาไทย (Thai) Terminology

## คำศัพท์หลัก
| ไทย | EN | คำอธิบาย |
|---|---|---|
| เช็คอิน | Check-in | ลงทะเบียนเข้าพัก |
| เช็คเอาท์ | Check-out | ออกจากที่พัก |
| การจอง | Reservation | จองห้องล่วงหน้า |
| การจอง/Booking | Booking | จองออนไลน์ |
| แขก | Guest | ผู้เข้าพัก |
| กุญแจห้อง | Room key | บัตรเข้าห้อง |
| อัตราเข้าพัก | Occupancy | เปอร์เซ็นต์ห้องเต็ม |
| ห้องว่าง | Vacancy | เปอร์เซ็นต์ห้องว่าง |
| ร้องเรียน | Complaint | แจ้งความไม่พอใจ |
| แผนกต้อนรับ | Reception | โต๊ะต้อนรับ |
| ความปลอดภัย | Security | ทีมรักษาความปลอดภัย |
| กล้องวงจรปิด | CCTV | ระบบกล้องรอบรู้ |
| ลัด้งเวียน | Patrol | รอบตรวจของเจ้าหน้าที่ |
| ฉุกเฉิน | Emergency | สถานการณ์ฉุกเฉิน |
| บัตรประจำตัว/หนังสือเดินทาง | ID/Passport | เอกสารยืนยันตัวตน |
| มัดจำ | Deposit | เงินคุ้มครอง |
| เช็คเอาท์ล่าช้า | Late checkout | ออกล่าช้ากว่ากำหนด |
| เช็คอินเร็ว | Early check-in | เข้าก่อนเวลาปกติ |
| ไม่มาตามนัด | No-show | จองแล้วไม่มา |
| มาเอง | Walk-in | มาโดยไม่จอง |

## ตัวย่อ
- CI = Check-in
- CO = Check-out
- OCC = อัตราเข้าพัก
- PAX = จำนวนแขก
- ETA = เวลาถึง
- ETD = เวลาออก

## การสะกดผิดที่พบบ่อย
- ✗ เช็ค-อิน → ✓ เช็คอิน
- ✗ ร้องเรียน → ✓ ร้องเรียน (เครื่องหมายถึง)
- ✗ อคุพอเครื่อง → ✓ Occupancy

## ประโยคตัวอย่าง
- แขกทำการเช็คอินที่แผนกต้อนรับเรียบร้อย
- อัตราเข้าพักอยูที่ 85% ห้องว่างน้อย
- ค่าเช็คเอาท์ล่าช้าถูกเก็บแล้ว
- ทีมลัด้งเวียนเข้าแทรกแซมฉุกเฉินทนที
- คืนนี้ไม่มีห้องสำหรับแขกมาเอง
