# Operasyon — Türkçe Terminoloji

## Temel Terimler
| TR | EN | Açıklama |
|---|---|---|
| Check-in | Check-in | Misafir giriş işlemi |
| Check-out | Check-out | Misafir çıkış işlemi |
| Rezervasyon | Reservation | Önceden oda ayırtma |
| Rezervasyon/Booking | Booking | Online ön rezervasyon |
| Misafir | Guest | Konaklayan kişi |
| Oda anahtarı | Room key | Odaya giriş kartı/belirteci |
| Doluluk oranı | Occupancy | Dolu odaların yüzdesi |
| Boş oda oranı | Vacancy | Boş odaların yüzdesi |
| Şikâyet | Complaint | Misafir memnuniyetsizliği bildirimi |
| Resepsiyon | Reception | Ön kabul gişesi/bankosu |
| Güvenlik | Security | Tesis güvenlik birimi |
| CCTV | CCTV | Kamera ile gözetleme sistemi |
| Devriye | Patrol | Güvenlik personeli turu |
| Acil durum | Emergency | Acil müdahale gerektiren hal |
| Kimlik/Pasaport | ID/Passport | Misafir kimlik belgesi |
| Depozito | Deposit | Teminat/garanti bedeli |
| Geç check-out | Late checkout | Gecikmeli çıkış |
| Erken check-in | Early check-in | Normal saatten önce giriş |
| Gelmeyen misafir | No-show | Rezerve edip gelmeyen |
| Walk-in (gelip-geçen) | Walk-in | Rezervasyonsuz gelen misafir |

## Kısaltmalar
- CI = Check-in
- CO = Check-out
- OCC = Occupancy
- PAX = Misafir sayısı
- ETA = Geliş saati
- ETD = Çıkış saati

## Yaygın Hatalı Yazılışlar
- ✗ Çek-in → ✓ Check-in
- ✗ Rezervasyon → ✓ Rezervasyon
- ✗ Depozit → ✓ Depozito
- ✗ Vakans → ✓ Vacancy

## Örnek Cümleler
- Misafir resepsiyonda check-in işlemini tamamladı.
- Doluluk oranı %85'e ulaştı; boş oda oranı düşük.
- Geç check-out ücreti tahsil edildi.
- Acil durumda devriye ekibi hemen müdahale etti.
- Walk-in misafir için oda mevcut değildi.
