# Operasyon — English Terminology

## Core Terms
| EN | TR | Description |
|---|---|---|
| Check-in | Check-in | Guest arrival registration |
| Check-out | Check-out | Guest departure process |
| Reservation | Rezervasyon | Booking a room in advance |
| Booking | Rezervasyon | Online or agent reservation |
| Guest | Misafir | Staying visitor |
| Room key | Oda anahtarı | Card or token for room access |
| Occupancy | Doluluk oranı | Percentage of filled rooms |
| Vacancy | Boş oda oranı | Percentage of empty rooms |
| Complaint | Şikâyet | Guest dissatisfaction report |
| Reception | Resepsiyon | Front desk area |
| Security | Güvenlik | Facility security team |
| CCTV | CCTV | Closed-circuit video surveillance |
| Patrol | Devriye | Security round/walk |
| Emergency | Acil durum | Urgent incident requiring action |
| ID/Passport | Kimlik/Pasaport | Guest identification document |
| Deposit | Depozito | Security hold amount |
| Late checkout | Geç check-out | Delayed departure past noon |
| Early check-in | Erken check-in | Arrival before standard time |
| No-show | Gelmeyen misafir | Reserved but did not arrive |
| Walk-in | Walk-in | Guest without reservation |

## Abbreviations
- CI = Check-in
- CO = Check-out
- OCC = Occupancy rate
- PAX = Number of guests
- ETA = Estimated time of arrival
- ETD = Estimated time of departure

## Common Misspellings
- ✗ Chekin → ✓ Check-in
- ✗ Reservasion → ✓ Reservation
- ✗ Occupency → ✓ Occupancy
- ✗ Vacansy → ✓ Vacancy

## Example Sentences
- The guest completed check-in at reception.
- Occupancy reached 85%; vacancy is low.
- Late checkout fee was charged.
- Security patrol responded to the emergency immediately.
- No walk-in rooms were available tonight.
